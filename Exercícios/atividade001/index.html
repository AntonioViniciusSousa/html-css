Descrevendo a Lógica de Programação

Lógica de Programação é o modo como se escrevem programas de computador através
de uma sequência de passos para executar uma ou várias funções, esta sequência é o algoritmo.

Algoritmo é basicamente uma “receita” para executar uma tarefa ou resolver um problema,
ele tem começo meio e fim, assim como a receita de bolo, se você seguir os passos descritos, terá um resultado.

Conectando os conceitos

Primeiramente encontramos o problema que queremos resolver. Depois pensamos num possível caminho para resolução.
Logo após, pensaremos em uma sequencia lógica utilizando um dos três tipos de algorítmos e no final traduzimos
esse passo-a-passo para uma linguagem de máquina. Essa linguagem de máquina se chama Linguagem de Programação.

Mas para que serve a programação?

Tão importante quanto saber ler e escrever, aprender a programar é crucial para tirar o máximo de proveito das
tecnologias. O pensamento homem-máquina é de um poder impressionante, contudo, saber utilizar e desenvolver
softwares que consigo mesclar as duas naturezas é um tanto quanto difícil, porém, quem não dominar essa
competência poderá ficar dependente de outras pessoas ou até mesmo ficar fora do mercado de trabalho.

Principais tópicos:

  1. Algoritmos e Fluxogramas;
  2. Variáveis e Tipos de Dados;
  3. Estruturas de Controle;
  4. Operadores;
  5. Funções e Procedimentos;
  6. Arrays (Vetores) e Estruturas de Dados Simples, dentre outros.
